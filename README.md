              Navigation Bar Documentation
1. Project Title
Responsive Navigation Bar with Logo
2. Objective
To design and implement a responsive navigation bar that adapts to different screen sizes using HTML and CSS without JavaScript.
3. Technologies Used
•	HTML5
•	CSS3
•	Flexbox
•	Media Queries
4. Components
 Navigation Container
The main wrapper that holds all navbar elements including logo, menu toggle, and navigation links.
 Logo Section
Displays the website logo image. Positioned on the left side of the navigation bar.
Navigation Links
Contains list items representing different sections of the website. Displayed horizontally on larger screens and vertically on smaller screens.
 Menu Toggle
Implemented using a checkbox input and label element. Controls the visibility of navigation links in mobile view.
5. Layout Design
•	Flexbox is used to align elements horizontally.
•	Space is distributed between logo and navigation links.
•	Gap property is used to maintain spacing between links.
6. Responsive Behavior
•	Media query is applied for smaller screen widths.
•	Navigation links are hidden in mobile view.
•	Menu icon becomes visible on smaller devices.
•	When activated, the navigation links are displayed vertically.
7. Styling Features
•	Background color applied to navigation bar.
•	Font styling applied to navigation links.
•	Hover effects improve user interaction.
•	Absolute positioning used for mobile dropdown layout.
8. Output
•	Desktop View: Horizontal navigation layout.
•	Mobile View: Collapsible vertical menu.
9. Conclusion
The responsive navigation bar successfully adapts to multiple screen sizes while maintaining structure and usability. The implementation ensures clean layout design and efficient performance without the use of JavaScript
